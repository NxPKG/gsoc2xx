---
title: "gsoc2 init"
description: "Switch between Gsoc2 projects within CLI"
---

```bash
gsoc2 init
```

## Description

Link a local project to your Gsoc2 project. Once connected, you can then access the secrets locally from the connected Gsoc2 project.

<Info>
This command creates a `gsoc2.json` file containing your Project ID.
</Info>
