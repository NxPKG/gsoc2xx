From local development to production, Gsoc2's language-specific SDKs provide the easiest way for your app to fetch back secrets on demand.

Prerequisites:

- Have a project with secrets ready in [Gsoc2 Cloud](https://app.gsoc2.com).
- Create an [Gsoc2 Token](/documentation/platform/token) scoped to an environment in your project in Gsoc2.

## Usage

Follow the instructions for your language use the SDK for it:

- [Node SDK](https://github.com/Gsoc2/gsoc2-node)
- [Python SDK](https://github.com/Gsoc2/gsoc2-python)

Missing a language? [Throw in a request](https://github.com/Gsoc2/gsoc2/issues).
